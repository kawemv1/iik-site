# Как добавить фотографии в галерею

## Инструкция:

1. Поместите ваши фотографии в папку `/public/photos/`
2. Назовите файлы: `photo1.jpg`, `photo2.jpg`, `photo3.jpg` и т.д.
3. Откройте файл `/src/components/PhotoGallery.tsx`
4. Обновите массив `photos`, заменив `/placeholder.svg` на `/photos/photo1.jpg` и т.д.

## Пример:

```typescript
const photos = [
  { id: 1, src: "/photos/photo1.jpg", alt: "Наш центр - фото 1" },
  { id: 2, src: "/photos/photo2.jpg", alt: "Наш центр - фото 2" },
  // ... и так далее
];
```

## Рекомендации:

- Используйте формат JPG или PNG
- Оптимизируйте размер изображений (рекомендуется до 1-2 МБ каждое)
- Рекомендуемое разрешение: 1200x1200px или больше
- Все фотографии будут автоматически обрезаны до квадратного формата
