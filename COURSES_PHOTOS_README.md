# Как добавить фото/постеры для курсов

## Инструкция:

1. Поместите фото/постеры курсов в папку `/public/courses/`
2. Назовите файлы:
   - `general-english.jpg` - для General English
   - `ielts.jpg` - для IELTS Preparation
   - `business-english.jpg` - для Business English
   - `academic-english.jpg` - для Academic English
   - `everyday-english.jpg` - для Everyday English
   - `grammar.jpg` - для English Grammar

## Рекомендации:

- Используйте формат JPG или PNG
- Оптимизируйте размер изображений (рекомендуется до 300 КБ каждое)
- Рекомендуемое разрешение: 1200x400px (широкое, горизонтальное)
- Все изображения должны быть одинакового разрешения для единообразия
- Если фото не найдено, будет показан градиент с иконкой курса

## Дизайн постеров:

Рекомендуется использовать:
- Яркие, привлекательные цвета
- Текст с названием курса (опционально)
- Образовательные элементы или иконки
- Единый стиль для всех постеров
