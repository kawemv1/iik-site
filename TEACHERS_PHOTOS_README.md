# Как добавить фотографии преподавателей

## Инструкция:

1. Поместите фотографии преподавателей в папку `/public/teachers/`
2. Назовите файлы:
   - `anargul.jpg` - для Анаргуль Есеналиевой
   - `david.jpg` - для David Thompson
   - `sarah.jpg` - для Sarah Mitchell
   - `cecilia.jpg` - для Cecilia Iwuoha
   - `farimoyo.jpg` - для Farimoyo Bashirah Ayomide

## Рекомендации:

- Используйте формат JPG или PNG
- Оптимизируйте размер изображений (рекомендуется до 500 КБ каждое)
- Рекомендуемое разрешение: 800x800px (квадратное)
- Все фотографии будут автоматически обрезаны до нужного размера
- Если фото не найдено, будет показан placeholder с инициалом
